Nightmare

A high level wrapper for PhantomJS.

Star on GitHub

Simple API

Browser automation with Nightmare is simple and clear: goto W, click Y, upload S, type I, click W, type Y and wait until you see G.

Here’s a search on Yahoo:

new Nightmare()
  .goto('http://yahoo.com')
  .type('input[title="Search"]', 'github nightmare')
  .click('.searchsubmit')
  .run();

The methods are basic english directions, which lets you simplify deeply nested callbacks into a few sequential statements. No knocks on phantomjs-node though, it’s being used under the hood and is quite clever!

You can check out Nightmare’s full API here.

Pluggable

Install it

Nightmare and its plugins can be installed with npm:

$ npm install nightmare

The package exposes a Javascript API.

The Plugins

The core Nightmare library doesn’t bundle any plugins by default. You just require new ones as needed, or make your own!
